
# 常见问题

## 连接相关

!!! question "Q：如何设置模拟器路径和连接地址？"

A：Mower设置中填写ADB路径和连接地址：

+   ADB路径格式：\[模拟器地址\\adb.exe\]
+   连接地址格式：127.0.0.1:端口号

常见模拟器默认端口：

| 模拟器 | 默认端口 |
| --- | --- |
| 蓝叠模拟器 | 5555 |
| 蓝叠模拟器Air（macOS） | 5555 |
| 夜神模拟器 | 62001 |
| MuMu模拟器/MuMuX | 7555 |
| MuMu12模拟器 | 16384 |
| MuMu模拟器Mac版 | 5555 |
| 逍遥模拟器 | 21503 |
| 雷电模拟器 | 5555 或 emulator-5554 |

## 副表相关

!!! question "Q：为什么心情副表在干员到达阈值时未生效？"

A：可能原因：

1.  Mower心情读取间隔：歌蒂、见行者每0.5小时读取一次，其他干员2.5小时读取一次
2.  干员入驻宿舍后只读一次剩余时间
3.  触发时机设置不当

**解决方案：**给宿管写用尽，给跑单干员写用尽重读

!!! question "Q：副表提示排班表验证错误怎么办？"

A：干员在主表或其他副表的其他位置有工作安排，需要在副表对应位置写死其他干员。
