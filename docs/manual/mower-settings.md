# Mower设置

## 基础设置

根据个人需求配置基本参数

## 宿舍黑名单

设置永不进宿舍的干员

**示例：**零心情工作干员Lancet-2

## 邮件提醒

配置接收任务通知的邮箱，通过问号图标查看授权码获取方式。通知等级分为：

+   INFO：基本信息
+   WARNING：警告信息
+   ERROR：错误信息

!!! info "高级别包含低级别信息，时差修正单位为小时。清空收件人字段可将邮件发送至本人邮箱"

## 森空岛相关

输入账号密码后，Mower可自动完成森空岛每日签到，并为仓库物品混合读取和无缝合成材料提供支持。
!!! success "当前正在开发的版本中已支持终末地签到，将在后续版本中加入"

## 无缝合成材料设置

位置：`全部设置` → `Mower设置` → `基建设置`

功能特点：

+   由"宿舍不养闲人回满"功能触发
+   需启用森空岛仓库读取功能
+   当目标材料数量低于上限且子材料数量高于下限时自动合成
+   支持多种材料合成策略

特殊说明：

+   九色鹿垫刀材料：消耗4心情以下的材料
+   九色鹿暴击材料：消耗4心情的材料
+   家具零件需用"家具零件\_碳"格式区分配方

**推荐合成路径：**`碳素` → `碳素组` → `家具零件\_碳素组` → `材料`

!!! info "合成截图保存在screenshot/workshop文件夹，条件不满足时发送邮件通知"

## 作业导入

从群文件或频道下载**图片/JSON格式**作业文件，通过`"排班编辑→导入排班"`加载

!!! warning "作业文件不包含主表黑名单列表，具体限制需参考作业说明或咨询作者。"

## 葛朗台跑单

跑单机制：但书、龙舌兰等干员的技能效果仅在订单生成瞬间生效。在贸易站安排高效干员，在订单即将结束时换上特殊干员，新订单生成后换回原班组的操作。

**配置方法：将跑单干员置于贸易站替换组首位**

选项说明：

+   启用葛朗台跑单：Mower会在跑单前数分钟（前置延时）准备干员，并在新单生成前数秒（缓冲时间）放入贸易站
+   禁用葛朗台跑单：干员立即放入贸易站并使用无人机加速

!!! info "网络或配置不佳时建议延长前置时间。"

## 心情阈值

干员或班组平均心情达到此百分比时，Mower会在高优位置充足时安排休息。实际心情值计算公式：

`实际心情值 = 心情下限 + 心情阈值 × (心情上限 - 心情下限) `

**示例：**阈值设为65%，夕在感知模式下的实际心情值为`12 + 0.65 × 12 = 19.8`

## 急救阈值

常规流程：干员到达心情阈值时若高优位充足则正常休息。

急救模式：当整组心情低于"心情阈值×急救阈值"且宿舍有空位时，强制安排休息。

设为0时禁用急救模式。

## 宿舍不养闲人

自动移出已回满的干员，替换为待回复干员，保持宿舍满员以触发依赖型技能（如黑键、乌有、迷迭香）。启用后会增加任务频率。

## 肥鸭防呆与肥鸭阈值

+   开启：所充干员均大于90%则肥鸭待机30分钟
+   关闭：所充干员均大于肥鸭阈值则优先充能心情最低者

## 宿舍优先级排序

某些布局中上层非VIP位回复能力可能高于下层VIP位，与默认逻辑冲突时可手动设置优先级顺序。更换宿舍布局后需更新此设置。
