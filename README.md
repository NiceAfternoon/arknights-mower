## 功能介绍

- 基建：跑单、按心情动态换班、自动合成材料；
- 森空岛：签到、仓库读取；
- 日常：公招、邮件、线索、清理智；
- 大型任务：生息演算、隐秘战线；
- 签到：五周年月卡、限定池每日一抽、矿区、孤星领箱子、端午签到……
- 调用 maa：肉鸽、保全。
